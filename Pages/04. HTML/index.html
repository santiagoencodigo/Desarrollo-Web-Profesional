Glosario de Términos Web

HTML
HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar para crear páginas web.
CSS
Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje utilizado para estilizar la presentación de documentos HTML.
JavaScript
Lenguaje de programación que permite crear contenido dinámico e interactivo en páginas web.
