Desarrollo Web Profesional

Proyecto personal en desarrollo, orientado a la consolidación de fundamentos del desarrollo web a través de una ruta de formación progresiva, basada en proyectos prácticos y documentación técnica.

Actualmente el proyecto aborda:

  • Fundamentos de la Web y funcionamiento de Internet.
  • Desarrollo frontend con HTML y CSS mediante proyectos prácticos.
  • Aplicación de fundamentos de JavaScript para la construcción de aplicaciones web interactivas.
  • Introducción a herramientas de desarrollo como Git, GitHub y la terminal.
  • Consolidación de bases de ingeniería de software y buenas prácticas.

Todo el proceso de aprendizaje, junto con la documentación y los proyectos desarrollados, se encuentra disponible en el repositorio: github.com/santiagoencodigo/Desarrollo-Web-Profesional
